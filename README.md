# NodeJS Chat

Chat powered by NodeJS, SocketIo and VueJS

## Prerquisites

- Node
- Redis-Server

## Start the project

This will launch `redis-server`, the websocket server and the websocket client.

``` bash

npm start

```

Open [http://localhost:8080/#/](http://localhost:8080/#/)
